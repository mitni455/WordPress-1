Back to Top ↑Introduction

This theme is using the newest HTML5 and CSS3 standards and will therefore not work with Internet Explorer 6 and 7. Everything up from Internet Explorer 8 and all major browsers like Firefox, Safari, Chrome and Opera have been tested successfully. It is however possible that minor glitches will appear in older versions of these browsers. The functionality however should always be given. Please also keep in mind that a few visual effects were made in CSS3 so the viewing experience might differ in older browsers. This theme will furthermore only work with Wordpress 3.4 or higher.

Back to Top ↑Installation

In order to install this theme you must have a working version of Wordpress 3.4 already installed. For information on how to install Wordpress please read the codex http://codex.wordpress.org/Installing_WordPress.

FTP

Use a FTP program of your choice to upload the non-zipped folder "timeline" from the theme package to "/wp-content/themes/". After that just navigate to "Appearance → Themes" and activate the theme.

Wordpress

Navigate to "Appearance → Themes → Install Theme → Upload", click on "Choose File" and choose the file "timeline.zip" from the theme package. If the upload and installation was successful you will be able to activate the theme instantly.

Back to Top ↑Setup

Please note that the theme will look quite empty with the placeholder content provided by Wordpress. Don't worry though, as long as you follow this guide it will look exactly as in the theme preview.

Import

If you rather want to skip the manual setup you can automatically import almost all content as shown in the theme preview. You are only required to change a few details after the import is finished.

First navigate to "Appearance → Theme Options" and click on "Import" in the upper right corner next to the logo. Select the file "serifly_timeline_theme_options_export.1352203015.txt" from the import folder of the theme package and confirm. This will set some small things in the theme options to get you started.

The next step is to install the WordPress importer. Navigate to "Tools → Import", click "WordPress" and install the plugin. As soon as the installation is finished you can select an import file to upload. Select the file "timeline.wordpress.2012-11-06.xml" located in "import" in your theme package. Don't import the user but instead assign all posts to an existing one and check the box to download all attachments. This process can now take some time depending on your connection as all timeline images will be retrieved from my server.

Now that everything has been imported, navigate to "Appearance → Menus" and select the menu "Header" as your "Primary Menu". If you skip this step the header layout will break on your site.

For the last step go to "Settings → Reading", check "A static page" under "Front page displays" and choose the page "Work" as your front page and "Blog" as your posts page.

The only part you will have to set up manually now are the sidebars and its widgets. Unfortunately they cannot be imported with the default WordPress importer.

Timeline

Navigate to "Pages → Add New", insert a title and choose the template "Timeline" from "Page Attributes". Don't put any content on this page and leave the hero unit empty as well. To set up this page as your homepage, navigate to "Settings → Reading" and select the page you just created under "Front page displays" as your static front page.

Now go to "Timeline → Add Item" to create your first timeline post. Creating a timeline post basically works exactly like creating a normal blog post. Enter a title and some content in the editor and insert a more tag, if you want to display the "View Project" link. Everything underneath the more tag will only be shown on the item page and not on the timeline page. Now scroll down and upload the images you want to show with the created item. Click on "Upload Image · 1", upload your image, select "Full Size" and click "Insert into Post". You will see your picture being displayed in the little preview area on the left. Now if you want you can enter a YouTube or Vimeo link into "Optional Link" to embed it when a user clicks on the image. By default the full size of the image will be shown on click.

After you have selected the images you want to show you can add a few tags to your item. Please note that only one tag can be used if you set your timeline "Navigation Filter" to "Tags". This setting can be changed in the theme options.

To bring your items in order, all you have to do is changed the "Publish Date" in the box labeled "Publish".

Regular Page

Add a new page and this time choose the template "Default Template" from "Page Attributes". This template will display the sidebar "Page Sidebar" which can be set up in the widget area of your Wordpress installation.

Full Width Page

Again navigate to "Pages → Add New", insert a title and choose the template "Full Width" from "Page Attributes". This template will never display a sidebar no matter if you set it up or not.

Blog

Navigate to "Pages → Add New", enter a title and choose the "Default Template" from the "Page Attributes". Navigate to "Settings → Reading" and select the page you just created under "Front page displays" as a static page to show your posts.

If you want to customize the sidebar used for the blog, navigate to "Appearance → Widgets" and set up the "Main Sidebar".

Contact Page

To create a contact page navigate to "Pages → Add New", enter a title and choose "Contact" from the "Page Attributes". You can change the position of the contact form, the sending confirmation message, an alternative email address and the captcha in the theme options under "General Options".

Back to Top ↑Theme Options

Using the theme options panel you can easily customize your site. It includes basic functions like adding a logo, changing the footer text and some color modifications as well as some detailed options.

General Settings

The "General" section lets you edit and modify your favicon, the navigation label shown on mobile devices, different options for the contact form, a custom feed and your tracking code or any other Javascript you want to attach to the site.

Header Options

In the "Header" section you can upload a logo or enter a text logo and disable the blog search field and if required replace it with a little link including one of four icons.

Footer Options

The "Footer" options allow you to enter your own line instead of using the default copyright notice and links.

Timeline Options

The "Timeline" section allows you to change your sites timeline to the tag filtered view. This means that all timeline items won't be shown ordered by date but instead by the tags you are using. You can also enter your own more tag label and add content to the small info box shown in the lower right corner.

Blog Options

In the "Blog" options you can set what should be displayed in the header of every post, shorten the name of the month to three letters and enter your own more tag label.

Styling Options

The "Styling" section lets you edit the color of all links displayed on the page, the background color of your header and some other minor changes like displaying the footer in grey instead of white.

Social Networks

In "Social Networks" you can enter the urls to your profiles for some very popular services. Those links will then be shown in the header with a small icon. If you are missing a service you are free to request it and I will try to include it in the next update.

Back to Top ↑Sidebars

You can set up four different sidebars for this theme, one is the sidebar for the blog content, one for content pages, one only for the contact page and one sidebar is used for the large footer. Navigate to "Appearance → Widgets" to set up your sidebars.

Back to Top ↑Menus

When using this theme you should always create custom menus for each purpose. Navigate to "Appearance → Menus" and create a new menu. Add the pages you want for the main navigation and select it as your "Primary Menu".

Back to Top ↑Widgets

The theme supports the basic Wordpress widgets and includes two new ones. The "Latest Timeline Item" widget will display the last item you have added to your timeline. Please note that it has to contain at least one image. The "Twitter Feed" widget will display the number of tweets you set from any given profile.

About Widget

Create a new text widget for the sidebar "Large Footer" and paste the following into the text box. Please do not check "Automatically add paragraphs". You can use the icon classes "iconEmail", "iconPhone", "iconChat", "iconUser" and "iconLocation".

<p>Your description goes here.</p>
[separator_line]
[one_half]
<p class="iconLocation">
	765 Doe Boulevard,<br />
	Rochester, NY 14621<br />
	United States
</p>
[/one_half]
[one_half_last]
<p>
	<span class="iconPhone">+1.234.567.8901</span><br />
	<a class="iconEmail" href="mailto:hello@example.com">hello@example.com</a><br />
	<a class="iconChat" href="http://serifly.com">Chat with us</a>
</p>
[/one_half_last]

Back to Top ↑Shortcodes

This theme has a variety of custom shortcodes to make the process of creating the pages easier. If you are not sure how to use those shortcodes you can import the demo content and have a look at the page "Shortcode".

Shortcode Browser

Every post, timeline and page content editor will display a little box on the upper right corner with all available shortcodes. Just focus the visual or HTML editor, click on the shortcode you want to use and it will be pasted automatically to the position of your text cursor.

Break

If required at some point of your content you can insert a break instead of entering a new line. This is especially helpful in text widgets.

[break]

Separator

The separator is a horizontal space which equals a four line breaks spacing.

[separator]

Separator Line

The separator line is a horizontal space which equals a eight line breaks spacing and displays a little border.

[separator_line]

Lists

Due to the functions of this theme you have to wrap a shortcode around every list you create.

[list]
	Visually created list
[/list]

or for widgets

[list]
	<ul>
		<li></li>
	</ul>
[/list]

You can choose from three different types of lists, the default look showing a small dot, a style with ticks and a style with crosses.

[list][/list]
[tick_list][/tick_list]
[cross_list][/cross_list]

Testimonial

This shortcode shows a little testimonial bubble with the customers name and information.

[testimonial author="John Doe, Example Inc"]Lorem ipsum dolor sit amet.[/testimonial]

Button

The default button color is the link color. You can set the attribute "style" to "green", "grey" or "light" and the attribute "size" to "auto" so the button won't be stretched to fit the content.

[button url="http://serifly.com" style="green" size="auto"]Label[/button]

Map

To show a map on your site use the following code. Be as precise as possible with your address.

[map location="New York"]

Info Box

Wraps your content in a box. Can hold any content.

[info_box][/info_box]

Video

To embed a video on your site use the following code. Either Youtube or Vimeo video url.

[video url=""]

Columns

You can specify up to four columns next to each other. To do so use the appropriate size. The last column of a row must always have _last attached to its tag.

[one_half][/one_half]
[one_third][/one_third]
[two_third][/two_third]
[one_fourth][/one_fourth]
[three_fourth][/three_fourth]

Tabs

To create a tabbed navigation and box use the following. You can put almost anthing inside the tab content (i.e. columns).

[tab_wrapper]
	[tab label="Tab 1 Label"]
		Content Tab 1
	[/tab]
	[tab label="Tab 2 Label"]
		Content Tab 2
	[/tab]
[/tab_wrapper]

Pricing Plans

Using the column layout you can easily set up pricing tables. Please have a look at the demo content included with this theme to learn more.

Use the column names "one_half" for two plans, "one_third" for three and "one_fourth" for four plans.

[pricing_plans]
	[one_half]
		[pricing_plans_header title="" description="" price="" term=""]
		[pricing_plans_features]
		<ul>
			<li>HTML or visually created list</li>
		</ul>
		[/pricing_plans_features]
		[pricing_plans_details]
		<ul>
			<li>HTML or visually created list</li>
		</ul>
		[/pricing_plans_details]
		[pricing_plans_order][button url="" style=""][/button][/pricing_plans_order]
	[/one_half]
	[one_half_last]
		[pricing_plans_header title="" description="" price="" term=""]
		[pricing_plans_features]
		<ul>
			<li>HTML or visually created list</li>
		</ul>
		[/pricing_plans_features]
		[pricing_plans_details]
		<ul>
			<li>HTML or visually created list</li>
		</ul>
		[/pricing_plans_details]
		[pricing_plans_order][button url="" style=""][/button][/pricing_plans_order]
	[/one_half_last]
[/pricing_plans]

Tables

If you want to use a regular table use following shortcodes.

[table]
	[table_title]Table[/table_title]
	[table_row]
		[table_cell_header]Plan Name[/table_cell_header]
	[/table_row]
	[table_row]
		[table_cell]Table Cell[/table_cell]
	[/table_row]
	[table_row]
		[table_cell_price label="Price Label"]Price[/table_cell_price]
	[/table_row]
	[table_row]
		[table_cell][button link=""]Button[/button][/table_cell]
	[/table_row]
[/table]
